## Installation

To set up the project locally:

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/your-portfolio.git
    ```
2. Navigate to the project directory:
    ```bash
    cd your-portfolio
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
